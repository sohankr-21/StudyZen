# 🌿 StudyZen

**StudyZen** is a smart study tracking web application designed to help students stay consistent, track their study sessions, and improve productivity through a clean and aesthetic dashboard.

---

## 🚀 Features

* 📚 Add and manage study sessions
* ⏱️ Track time spent on each subject
* 📊 Dashboard with study statistics
* 🔥 Clean sidebar layout (modern UI)
* 🌸 Pastel-themed aesthetic design
* 💾 Data stored using browser local storage

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---
## 📂 Project Structure

```
StudyZen/
│── index.html
│── styles.css
```

---

## ⚙️ How to Run

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser
   OR use **Live Server** in VS Code
